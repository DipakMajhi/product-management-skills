# pm-market-research

User research and competitive intelligence skills for product managers.

## Skills (4)

- **user-personas** — Create evidence-based user personas with JTBD statements and anti-patterns.
- **customer-journey-map** — Map the end-to-end customer journey with emotion curves and opportunity areas.
- **competitor-intelligence** — Build deep competitor profiles with win/loss analysis and battlecard notes.
- **sentiment-analysis** — Extract themes, churn signals, and advocacy signals from user feedback.

## Commands (4)

- `/pm-market-research:research-users` — Build user personas and map the customer journey.
- `/pm-market-research:journey-map` — Map the end-to-end customer journey for a persona.
- `/pm-market-research:profile-competitor` — Build a deep competitor profile.
- `/pm-market-research:analyze-feedback` — Analyze user feedback for sentiment themes and insights.
