# Pricing

This skill routes to **pricing-strategy**, which covers the full pricing decision lifecycle.

**What pricing-strategy covers:**
- Value quantification (economic, emotional, displacement)
- Value metric selection (HOPE framework)
- Willingness-to-pay research methods (Van Westendorp, Gabor-Granger, MaxDiff, Conjoint)
- Pricing model selection (value-based, usage-based, hybrid, outcome-based, freemium)
- Good-Better-Best packaging architecture
- AI/ML product pricing considerations
- B2B and B2C psychology tactics
- Price localization (PPP)
- NRR and expansion design
- Pricing experimentation

**Routing by context:**
- First-time pricing → Start with value quantification and value metric selection
- Revising existing pricing → Start with WTP research and competitive benchmarking
- Packaging/tier design → Start with Good-Better-Best architecture
