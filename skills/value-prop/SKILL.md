# Value Prop

This skill routes to **value-proposition**, which provides a complete value proposition framework combining JTBD theory, Strategyzer Value Proposition Canvas, and April Dunford's positioning methodology.

**When to use this:**
- Defining your value proposition from scratch for a new product or segment
- Sharpening a vague or generic value prop ("we save you time and money")
- Preparing positioning for a launch, investor pitch, or sales enablement
- Answering "what's the value proposition of this product?" in PM interviews

**Quick approach guide:**
- Unknown customer needs → Start with JTBD discovery: what job is the customer hiring the product to do?
- Known customer needs, unclear differentiation → Strategyzer Value Proposition Canvas: map pains, gains, and products/services
- Clear differentiation, need a statement → April Dunford's framework: unique attributes → value → segment → category
- Need a positioning statement → Geoffrey Moore's template: "For [segment] who [need], [product] is a [category] that [benefit], unlike [alternative]"

**What value-proposition covers:**
- Jobs-to-be-Done theory (Clayton Christensen + Bob Moesta): functional, social, and emotional jobs; forces of progress
- JTBD interview technique: extract the real job from customer language ("when I hired this product, I was trying to…")
- Strategyzer Value Proposition Canvas: customer jobs / pains / gains → products/services / pain relievers / gain creators
- Pain vs. gain distinction: pains are problems, frustrations, and risks; gains are outcomes and aspirations
- Pain relievers vs. gain creators: how your product maps to both
- April Dunford's 5-step positioning framework: competitive alternatives → unique attributes → value → target customers → category
- Value quantification: economic value (ROI, cost savings), emotional value (confidence, status), displacement value (what they used to pay for)
- Value proposition statement formats: positioning statement, elevator pitch (30 sec), tagline
- Multi-segment value props: how to adapt the value prop for different buyer/user/champion roles
- Anti-patterns: feature-listing, jargon, "faster/cheaper/better" without specifics, value props that apply to competitors too

**The skill will:**
1. Ask about the target customer, problem, and competitive alternatives
2. Extract the functional, social, and emotional jobs being done
3. Map pains and gains using the Strategyzer Canvas
4. Apply April Dunford's framework to identify unique value
5. Write the value proposition statement in 2-3 formats
6. Test the value prop: would a customer say "that's exactly what I need" or "so does everyone else"?
