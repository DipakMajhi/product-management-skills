# Prioritize

This skill routes to **prioritization**, which covers 12 prioritization frameworks with selection guidance.

**Quick framework guide:**
- Large backlog (50+ items), limited data → ICE Score
- Need to justify to stakeholders with numbers → RICE Score
- Scoping a release: must-haves vs. nice-to-haves → MoSCoW
- Time-sensitive decisions, cost of delay matters → WSJF
- Discovery work, prioritizing problems not features → Opportunity Solution Tree
- Portfolio allocation across time horizons → Three Horizons
- Small list, need honest forced ranking → Stack Ranking

**The skill will:**
1. Confirm what is being prioritized and the decision context
2. Recommend the right framework
3. Apply it with scoring for each item
4. Flag counterintuitive results and explain why
5. Identify dependencies affecting sequencing
6. Present trade-off analysis
