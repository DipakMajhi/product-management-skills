# Growth Loop Designer

Growth loops are circular, compounding systems where the output of one cycle becomes the input of the next - unlike funnels, which are linear and do not compound.

## Types of Growth Loops

Apply this skill to: $ARGUMENTS


### Viral / Word-of-Mouth Loop
User gets value from product to User invites or refers others to New users join to Each creates value and refers more
Examples: Slack (invite teammates), Calendly (share scheduling link), Loom (send video)

Key metric: Viral coefficient (k) = invites sent per user x conversion rate of invites
If k > 1: exponential growth. If k < 1: positive but linear growth addition.

### Content Loop
User creates content to Content is indexed and discovered to New users find content to Some create more content
Examples: YouTube, Reddit, Stack Overflow, Substack

### Data Network Effect Loop
More users to More data to Better product intelligence to Better product to More users
Examples: Google Search, Waze, Duolingo (adaptive learning), Spotify Discover Weekly

### Marketplace Loop
More buyers to More sellers attracted to More inventory to More buyers attracted
Examples: Etsy, Airbnb, App Store

### Product-Led Loop (PLG)
User achieves success to Shares output with non-users to Non-users see value to Convert to users
Examples: Figma (share designs), Airtable (share base), Notion (share page)

## Loop Design Process

1. **Identify the value moment**: What is the single most valuable thing a user does or gets in your product?
2. **Find the natural share/spread**: Does that value moment naturally cause the user to spread the product? How?
3. **Map the loop**: Draw the circular chain from user action to new user acquisition
4. **Identify the break points**: Where does the loop lose efficiency? (Share rate, conversion rate, activation rate)
5. **Design interventions**: What product changes would improve the break points?
