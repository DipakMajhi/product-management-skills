# pm-data-analytics

Data analytics skills for product managers who work with data.

## Skills (3)

- **sql-for-pms** — Generate SQL queries for product analytics in BigQuery, PostgreSQL, or MySQL.
- **ab-testing** — Design A/B tests with sample size calculation and analyze results with Ship/Iterate/Stop recommendations.
- **cohort-analysis** — Build cohort retention analyses and diagnose the root cause of retention problems.

## Commands (3)

- `/pm-data-analytics:write-query` — Generate a SQL query for a product analytics question.
- `/pm-data-analytics:analyze-test` — Design or analyze an A/B test.
- `/pm-data-analytics:analyze-cohorts` — Build or interpret a cohort retention analysis.
