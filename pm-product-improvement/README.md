# pm-product-improvement

Product improvement skills for retention, funnel optimization, and post-launch iteration.

## Skills (3)

- **retention-analysis** — Diagnose retention problems and design targeted interventions.
- **funnel-optimization** — Identify funnel bottlenecks and build an experiment roadmap to improve conversion.
- **product-iteration** — Build a prioritized post-launch iteration plan from feedback and analytics data.

## Commands (4)

- `/pm-product-improvement:improve-retention` — Diagnose and design a plan to improve product retention.
- `/pm-product-improvement:optimize-funnel` — Analyze and optimize a product funnel for higher conversion.
- `/pm-product-improvement:iterate` — Design a post-launch iteration plan for an existing feature.
- `/pm-product-improvement:improve-product` — Answer "how would you improve [product]?" with a structured PM framework.
